The goal of this portfolio is to allow the reader to view my projects and find out more infomation about me with a simple layout to not overcomplicate or confuse the reader. I achieved this by making my web layout simplistic and easy to understand in a logic order. A feature of my portfolio webpage is that it is responsive, making the layout change depending on the size of the window. This allows users to view the webpage on phones or tablets and was achieved using media queries and styling changes.
